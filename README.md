# Projectile Shooting Game

# Description
This is a simple projectile shooting game.
You can control a blue circle at the center of the screen and shoot projectiles at colorful enemies that appear from all side.
The goal is to survive and score points by destroying enemies.

# Features
- Player
- Colorful enemies
- Score tracker
- Start game button
- Explosion effects
- Responsive design (works on different screen sizes)

# Tools
- HTML5
- Tailwind CSS
- JavaScript
- CSS
- Canvas API
- GSAP

# Challenges
- A challenge was figuring out how to make enemies spawn randomly from all sides of the screen.
- And how to handle collisions between the projectile and enemies.

# Learned
- How to use a Canvas API to draw shapes and animate them.
- How to apply projectiles, player movement, and score in JS.
- Adding visual effect like explosions with particles.
- Use Tailwind CSS to improve UI.
